VRChatの起動モードを指定して起動するやつ

## コレは何？
VRChatのホームページやURL経由でVRChatのインスタンスにJoinするとき、  
VR HMDが接続されているとVRモードで起動してしまいます。  
本プログラムはVR HMDが接続されていてもデスクトップモードで起動するか、  
VRモードで起動するか選べるようにするためのプログラムです。  

## 使い方
同梱してある「ReadMe　最初に読んでください」を読んでください。
